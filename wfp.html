Grand Theft Auto Vice City Widescreen Fix

http://www.rockstargames.com/vicecity/

Supported exe version: Any | v1.0 recommended

Installation: Unpack content of archive to GTA VC root directory. Also download archive of widescreen frontend textures or set FrontendAspectRatio option to 4:3 or 1:1 if you want to keep original textures. You may use content of scripts folder without d3d8.dll if that's necessary(copy to mss or root dir). d3d8.dll also contains d3d8to9 wrapper activated by default. It is used for mods that require rwd3d9.dll, like transparent menu (available in dropdown menu). Check the .ini file to get more info about available features.

  • Widescreen frontend textures
  • Transparent menu(for v1.0 exe only)
  • SaveLoader
